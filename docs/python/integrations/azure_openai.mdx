## Technical Overview
The SDK seamlessly tracks the calls made to the openai library in Python by encapsulating its methods. When an OpenAI method is invoked, the response is returned as usual. However, during this process, pertinent information is captured and sent asynchronously to nebuly, ensuring your code's performance remains unaffected. In this section, you will find a comprehensive list of the tracked methods, along with the corresponding data collected for each API call. Our goal is to provide clarity while maintaining the efficiency of your code execution.
